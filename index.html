Q1. Explain br tag?

In HTML, the <br> tag is used to insert a line break or a new line within a paragraph or other text element to create more readable content.. It is a self-closing tag, which means it doesn't require a closing tag like other HTML tags.

Q2. Explain how many different types of lists we have in html?

There are three main types of lists that you can create in HTML:

  1. Ordered Lists (<ol>): Ordered lists are used to display a list of items in a specific order. Each item is listed with a number or letter that indicates its position in the list. The ordered list is created using the <ol> tag and each list item is created using the <li> tag.
  2. Unordered Lists (<ul>): Unordered lists are used to display a list of items that are not in a specific order. Each item is listed with a bullet point or other symbol that indicates its position in the list. The unordered list is created using the <ul> tag and each list item is created using the <li> tag

Q3. What are semantic tags?

Semantic tags are HTML tags that are used to describe the content and structure of a web page in a more meaningful way for both humans and search engines. They provide contextual information about the content within the tag, making it easier for web developers and search engines to understand the purpose of each element on the page.

  • <header>: This tag is used to define the header section of a web page, typically containing the site logo, navigation menu, and other introductory content.
  • <nav>: This tag is used to define the navigation section of a web page, typically containing links to other pages on the site.
  • <main>: This tag is used to define the main content area of a web page.

Q4. Write code to create an ordered list with roman numbers?

Ordered list in html is used to create list elements which are following some specific order. Its default type is number or “1”. Other types which it supports are “I” , “i”, “A”, “a”.

            <ol type="I">
                <li>Apple</li>
                <li>Orange</li>
                <li>Banana</li>
            </ol>
Output:
  1. Apple
  2. Orange
  3. Banana

Q5. Write code to create an unordered list with circle?

Unordered list in html is used to create list elements which are random in fashion. Its default type is bullet or “disc”. Other types which it supports are “circle” , “square”, “none” 

        <ul type="circle">
            <li>First</li>
            <li>Second</li>
            <li>Third</li>
            </ul>
Output:
  • First
  • Second
  • Third
